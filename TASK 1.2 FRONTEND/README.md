# Task 1.2 — E-Commerce Admin Dashboard
A static admin dashboard layout for an e-commerce platform built using semantic HTML5 and CSS Grid.

## Tech Used
- HTML5 (semantic tags: header, aside, main, section, article, nav, table)
- CSS3 (Grid layout, Flexbox, CSS Variables)


## Features
- Sidebar navigation
- stat cards (Revenue, Orders, Customers, Avg Order Value)
- Monthly revenue bar chart (CSS only)
- Top categories donut chart (SVG)
- Recent orders table with status badges

## How to Run
Open `dashboard.html` in browser
